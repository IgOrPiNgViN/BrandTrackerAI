# Руководство по внесению вклада

Спасибо за интерес к проекту! Мы рады любому вкладу.

## Как внести вклад

1. **Форкните репозиторий**
2. **Создайте ветку для вашей функции** (`git checkout -b feature/AmazingFeature`)
3. **Внесите изменения** и добавьте тесты
4. **Убедитесь, что все тесты проходят**
5. **Закоммитьте изменения** (`git commit -m 'Add some AmazingFeature'`)
6. **Запушьте в ветку** (`git push origin feature/AmazingFeature`)
7. **Откройте Pull Request**

## Стандарты кода

- Используйте PEP 8 для Python кода
- Добавляйте комментарии к сложным функциям
- Пишите тесты для нового функционала
- Обновляйте документацию при необходимости

## Сообщения о проблемах

При создании issue, пожалуйста:

- Опишите проблему подробно
- Укажите шаги для воспроизведения
- Приложите логи ошибок (если есть)
- Укажите версию Python и ОС
